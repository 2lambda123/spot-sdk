get /runs/
Query a collection of runs. (runsGet)
Returns a list of run resources based upon provided query parameters. A run describes a period of mission playback or teleoperation.

Query parameters

startTime (optional)
Query Parameter — Include runs with a start_time greater than or equal to this value. The value should be in isostring format. For e.g: '2023-06-05T19:29:35.066Z' default: null format: date-time
endTime (optional)
Query Parameter — Include runs with a start_time less than this value. The value should be in isostring format. For e.g: '2023-06-05T19:29:35.066Z' default: null format: date-time
modifiedAt (optional)
Query Parameter — Include runs with a modified_at time greater than this value. The value should be in isostring format. For e.g: '2023-06-05T19:29:35.066Z' default: null format: date-time
missions (optional)
Query Parameter — Only include runs performing one of the provided missions. The value should be a single or a comma-separated list of mission names. default: null
robots (optional)
Query Parameter — Only include runs performed by the provided robots. The value should be a single or a comma-separated list of robot hostnames. default: null
types (optional)
Query Parameter — Only include runs of this type. The value can be either 0 or 1 corresponding to mission and teleoperation respectively. default: null
robotHostname (optional)
Query Parameter — Include runs performed by a robot with this hostname default: null
robotNickname (optional)
Query Parameter — Include runs performed by a robot with this nickname. default: null
robotSerial (optional)
Query Parameter — Include runs performed by a robot with this serial. default: null
limit (optional)
Query Parameter — Only return a number of runs up to this value. default: 20
offset (optional)
Query Parameter — Skip this number of runs before returning. default: 0
orderBy (optional)
Query Parameter — Criteria for ordering results. default: newest
format (optional)
Query Parameter — The data format of the response. Set format to csv to download a csv file of summary run stats. By default, format is set to json which returns run_resources. default: json
